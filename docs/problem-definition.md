# Problem Definition

## Research Title

### Indonesian Title

**Analisis Performa Algoritma Machine Learning untuk Klasifikasi Jenis dan Tingkat Keparahan Cyberbullying pada Teks Bahasa Indonesia Menggunakan TF-IDF**

### English Title

**Performance Analysis of Machine Learning Algorithms for Cyberbullying Type and Severity Classification in Indonesian Text Using TF-IDF**

---

# 1. Research Topic

The main topic of this research is:

> **Natural Language Processing (NLP) and Machine Learning for Cyberbullying Classification in Indonesian Text.**

This research focuses on analyzing and comparing the performance of multiple Machine Learning algorithms in classifying cyberbullying-related Indonesian text.

The research uses **Term Frequency-Inverse Document Frequency (TF-IDF)** as the primary feature extraction method to convert textual data into numerical features that can be processed by Machine Learning algorithms.

The research focuses on two main classification objectives:

1. Cyberbullying Type Classification.
2. Cyberbullying Severity Classification.

---

# 2. Background

The rapid development of social media and digital communication platforms has made it easier for people to communicate, share opinions, and interact with one another. However, the growth of digital communication has also increased the occurrence of negative online behavior, including cyberbullying.

Cyberbullying refers to harmful, aggressive, abusive, threatening, or offensive behavior conducted through digital communication platforms. Unlike traditional bullying, cyberbullying can occur continuously, spread rapidly, reach a large audience, and remain accessible through digital platforms.

The detection of cyberbullying in Indonesian-language text presents several challenges. Digital communication commonly contains informal language, slang, abbreviations, misspellings, repeated characters, emojis, mixed language, and context-dependent expressions.

For example, the following expressions may have different meanings and severity levels:

```text
"Dasar bodoh"
```
