# Feature Engineering Run Guide

## Overview

This guide explains how to execute the TF-IDF feature engineering pipeline for the **Cyberbullying Type Classification** research project.

The pipeline transforms preprocessed Indonesian text into sparse numerical feature vectors for model training.

```
final_dataset_clean.csv → Split (80/20) → Fit TF-IDF (train only) → Transform → Save artifacts
```

---

## 6. How to Execute — Python Script

```bash
cd /home/zapp/Kampus/PM

python3 -c "
import sys; sys.path.insert(0, '.')
import os

from src.config.settings import CLEAN_DATASET_PATH, FEATURE_REPORTS_DIR
from src.config.tfidf_config import TfidfConfig
from src.data.load_dataset import load_dataset
from src.feature_engineering.tfidf_pipeline import (
    run_tfidf_pipeline, generate_feature_report, save_feature_report,
)
from src.feature_engineering.feature_statistics import (
    get_top_tfidf_terms, export_feature_statistics, export_feature_importance_preview,
)
from src.feature_engineering.feature_validator import validate_features
from src.feature_engineering.persistence import save_tfidf_artifacts

# Load
df = load_dataset(filepath=CLEAN_DATASET_PATH)

# Configure
config = TfidfConfig()

# Run pipeline
X_train, X_test, y_train, y_test, vectorizer, stats = run_tfidf_pipeline(df, config)

# Validate
validation = validate_features(X_train, X_test, y_train, y_test, vectorizer)
print(validation['summary'])

# Save artifacts
save_tfidf_artifacts(vectorizer, X_train, X_test, y_train, y_test, overwrite=True)

# Generate reports
top_terms = get_top_tfidf_terms(vectorizer, X_train, n=30)
report = generate_feature_report(stats, config, validation, top_terms)
save_feature_report(report, os.path.join(FEATURE_REPORTS_DIR, 'feature_summary.md'))
export_feature_statistics(stats, os.path.join(FEATURE_REPORTS_DIR, 'feature_statistics.csv'))
export_feature_importance_preview(top_terms, os.path.join(FEATURE_REPORTS_DIR, 'feature_importance_preview.csv'))

print('\n✓ Feature engineering complete.')
"
```

## 8. Troubleshooting

### ModuleNotFoundError: No module named 'sklearn'

```bash
pip install scikit-learn
```

### FileNotFoundError: final_dataset_clean.csv

Run the preprocessing pipeline first (notebook 02).

### ImportError: No module named 'src'

Run from the project root:

```bash
cd /home/zapp/Kampus/PM
```

### AssertionError: Missing text_clean column

The dataset must have a `text_clean` column from preprocessing. Run notebook 02 first.

---

## 9. Common Errors

| Error | Cause | Fix |
|-------|-------|-----|
| `FileNotFoundError` | Input CSV missing | Run preprocessing first |
| `ValueError: Missing column` | Wrong CSV format | Verify columns: text, label, text_clean |
| `ModuleNotFoundError` | Package not installed | `pip install scikit-learn scipy joblib` |
| `FileExistsError` | Output file exists | Set `overwrite=True` |
| `MemoryError` | Dataset too large | Set `max_features` in TfidfConfig |
